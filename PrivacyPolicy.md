# Privacy Policy

**Gemini Mode Selector**
Last updated: 2026-04-13

## Overview

Gemini Mode Selector is a Chrome extension designed to improve usability of the Gemini web interface by automatically selecting a preferred response mode and providing quick switching functionality.

We are committed to protecting user privacy. This extension does not collect, store, or transmit any personal data.

---

## Data Collection

Gemini Mode Selector does **not collect any personally identifiable information (PII)**.

The extension may access limited information within the active Gemini web page solely to perform its core functionality, such as:

* Detecting available response modes (e.g., Thinking, Fast)
* Applying the selected mode
* Displaying a small UI for mode switching

This information is processed locally in real time and is **not stored or transmitted**.

---

## Data Storage

The extension uses Chrome's `storage` API to store user preferences locally, including:

* Default mode selection
* Fallback behavior
* UI settings (e.g., language, quick switch visibility)
* Usage counters (e.g., number of switches)

All stored data:

* Remains **on the user's device**
* Is **never transmitted externally**
* Is **not accessible by the developer**

---

## Data Transmission

Gemini Mode Selector does **not send any data to external servers**.

There are:

* No APIs
* No analytics tools
* No tracking mechanisms

---

## Third-Party Sharing

No user data is shared with any third parties.

---

## Permissions Usage

The extension requests only the minimum permissions required:

* `storage`
  Used to store user preferences locally

* Host permission for `https://gemini.google.com/*`
  Required to interact with the Gemini web interface and apply mode selection

---

## Remote Code

The extension does **not use remote code**.
All scripts are bundled within the extension package.

---

## User Control

Users have full control over the extension:

* The extension can be disabled or removed at any time via Chrome settings
* All stored data is removed when the extension is uninstalled
